### How to Use?

1. Clone this repo -
2. 
   ```
   git clone https://github.com/EzazAA/Chrome-homepage.git
   ```

   or download the ZIP file
   
3. Go to the Directory
   
   ```
   cd Chrome-Homepage
   ```
   
4. Customize the `Index.html` file according to you
